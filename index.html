▸ find()

Convenience method. Dispatch to findByUrl or findByHash. If an hash is specified, it has priority over URL (depends on adapter implementation)

find(): Promise
Returns
Promise: Resolve with the shorturl object. Reject with error.
Example
adapter.find({ hash: 'XXXX' });
adapter.find({ url: 'http://domain.tld' });
adapter.find({ url: 'http://domain.tld', hash: 'XXXX' });
